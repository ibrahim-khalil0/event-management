###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<p>

#### Answer: B

<i>Here a variable is declared with let but no value is given to it. It is then reassigned to an object. We know that if we declare a variable with let, it needs to be resigned later. And then after console.log it an object is found.</i>
